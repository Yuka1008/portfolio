About わたしのこと

小泉　佑香 - Yuka Koizumi

1998年10月、神戸市生まれ。武庫川女子大学 食物栄養学科卒業後、管理栄養士とエステティシャンを経験。2024年6月、ものづくりで人を喜ばせたい想いからデジタルハリウッドSTUDIO三宮へ。Webデザインとコーディングを学び、多角的な経験を活かし、新たなクリエイティブに挑戦中。

Skills できること

  1. (01) Web Design

    Webサイト、LP、バナーなどの制作ができます。見た目の美しさはもちろん、情報設計にもこだわり、常にユーザー目線でデザインを進めています。細部にまで気を配り、使いやすさと魅力を両立するクリエイティブを提供します。

    Tools : Illustrator / Photoshop / Figma

  2. (02) Web Development

    HTML・CSSでデザインカンプを正確に再現し、JavaScriptで動きを加えることができます。命名規則を徹底し、誰が見ても理解しやすいコードを心がけています。レスポンシブ対応はもちろん、AIを活用した複雑なアニメーションも実装可能です。

    Tools : Visual Studio Code / WordPress
